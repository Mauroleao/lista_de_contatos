# Lista de Contatos

Aplicação web para gerenciamento de contatos desenvolvida com React, Redux e Styled Components.

## Funcionalidades

- Adicionar novos contatos com nome completo, e-mail e telefone
- Editar informações de contatos existentes
- Remover contatos da lista
- Buscar contatos por nome, e-mail ou telefone

## Tecnologias Utilizadas

- React
- Redux Toolkit
- Styled Components
- TypeScript

## Como Executar

No diretório do projeto, você pode executar:

### `npm install`

Instala todas as dependências do projeto.

### `npm start`

Executa a aplicação em modo de desenvolvimento.
Abra [http://localhost:3000](http://localhost:3000) para visualizar no navegador.

### `npm run build`

Cria uma versão otimizada para produção na pasta `build`.
